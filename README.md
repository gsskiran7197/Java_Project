# Java_Project
A simple Java-based Driver Management System that tracks driver records, vehicle categories, and travel distances. This project demonstrates core Java concepts like OOP, Encapsulation, ArrayLists, and Java 8 Stream API.



## What this project does:
- **Manages Driver Records**: Stores names, IDs, vehicle types (like car, lorry, auto), and total distance travelled.
- **Search Functionality**: Quickly find driver details using their unique Driver ID.
- **Filtering**: List all drivers belonging to a specific category (e.g., finding all "lorry" drivers).
- **Performance Tracking**: Identifies which driver has covered the most distance.

## Java Concepts Covered:
- **Object-Oriented Programming (OOP)**: Using classes and objects to represent real-world entities like `Driver`.
- **Encapsulation**: Using private variables and public getter/setter methods to protect data.
- **Collections Framework**: Using `ArrayList` to store and manage lists of drivers.
- **Java 8 Stream API**: Using modern Java features like `.stream()` and `.filter()` for cleaner and faster code.
- **Method Implementation**: Writing logic for searching, counting, and finding maximum values in a list.

## Project Structure:
- `Driver.java`: Defines the Driver object properties.
- `Travel.java`: Contains the main logic for managing and filtering driver data.
- `Tester.java`: A simple test file to run the project and see results.
